# QuizGrades
QuizGrades
A Java-based utility designed to process and analyze a set of academic scores. This program demonstrates fundamental programming constructs, including array management, iterative input processing, and multi-branch conditional logic.

Technical Highlights
Input Validation & Sentinel Control: Utilizes a while loop with a sentinel value (999) to allow for dynamic data entry lengths up to a fixed maximum of 10 records.

Array Management: Implements fixed-size array storage to track and display entered grades, ensuring data integrity throughout the execution lifecycle.

Statistical Computation: Performs real-time summation and average calculation, converting integer inputs to double precision for accurate grading metrics.

Grading Logic Architecture: Employs a multi-tier if-else if structure to map numerical averages to standard letter grades (A-F) based on a defined academic scale.

Technical Specifications
Language: Java (JDK 8+)

Key Concepts: Sentinel Loops, Array Indexing, Casting (Integer to Double), Conditional Mapping.
